# 1. 基础环境

  • 安装 NodeJs
  • 安装 hexo-cli : 全局安装 npm install hexo-cli -g

# 2. 初始化项目

  • 初始化项目文件夹 : hexo init blog
  • 安装 node 依赖: cd blog && npm install
  • 启动项目: hexo server -p port 指定本地启动端口，默认为 4000
