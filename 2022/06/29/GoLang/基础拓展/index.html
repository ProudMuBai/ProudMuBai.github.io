# 2. 原码，反码，补码

  1. 二进制的最高位是符号位: 0 表示正数，1 表示负数
  2. 正数的原码，反码，补码都一样
  3. 负数的反码 = 其原码的符号位不变，其他位取反 (0 ->1, 1->0)
    • 1 ===> 原码 (0000 0001) 反码 (0000 0001) 补码 (0000 0001)
    • -1 ===> 原码 (1000 0001) 反码 (1111 1110) 补码 (1111 1111)
  4. 负数的补码 = 其反码加一
  5. 0 的反码，补码 都是 0
  6. 在计算机运行时，都是一补码的方式来运算的 1+1 1-1 = 1+(-1)
